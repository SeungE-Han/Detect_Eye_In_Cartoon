# Detect_Eye_In_Cartoon
입력 이미지에서 인간의 눈과 유사한 영역을 찾아내기

조건: 눈은 가로방향 3w, 세로방향 w인 크기, 눈동자는 w X w크기의 정사각형이라고 가정함. 
목표: 눈의 시작좌표(좌상좌표)와 w를 찾아내어 빨간색 사각형으로 표시
